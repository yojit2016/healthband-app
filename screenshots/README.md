# Screenshots

This folder contains screenshots for the README.md file.

## Required Screenshots:

- `dashboard.png` - Screenshot of the main dashboard showing health metrics
- `emergency.png` - Screenshot of the emergency alert overlay
- `medications.png` - Screenshot of the medications tab
- `notifications.png` - Screenshot of the notifications tab

## How to Add Screenshots:

1. Run the app on an emulator or device
2. Navigate to each screen
3. Take screenshots
4. Save them in this folder with the exact filenames above
5. Commit and push to update the README

## Notes:

- Images should be PNG format
- Recommended size: 300-500px width for optimal README display
- Ensure screenshots show the app's UI clearly
